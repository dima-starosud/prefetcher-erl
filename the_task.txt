Тестове завдання: Реалізувати API для отримання курсів валют
Завдання:
Необхідно реалізувати веб-сервер для отримання курсів валют у форматі XML.
Веб-сервер має бути реалізований з використанням бібліотеки https://github.com/ninenines/cowboy
У момент запиту програма має виконати запит сервер-сервер на API ПриватБанку та отримати реальні курси валют у форматі JSON (https://api.privatbank.ua/p24api/pubinfo?json&exchange&coursid=5). Далі необхідно розпарсити JSON та зберегти отримані дані в таблицю ETS на 1 хвилину. Всі подальші запити протягом однієї хвилини мають брати дані з кешу таблиці ETS.
У відповідь на запит програма має повернути курси валют у форматі XML, для цього необхідно перетворити отримані дані з JSON у формат XML.

<exchangerates>
    <row>
        <exchangerate base_ccy="UAH" buy="43.30000" ccy="EUR" sale="44.30000" />
    </row>
    <row>
        <exchangerate base_ccy="UAH" buy="40.25000" ccy="USD" sale="40.85000" />
    </row>
</exchangerates>
